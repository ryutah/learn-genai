Google Cloud 2025年 動向レポート

AIが拓く、クラウドコンピューティングの新たな地平

2025年10月

エグゼクティブサマリー

2025年、Google CloudはAIを中核に据え、5つの主要領域で大きな進化を遂げます。

  • 1️⃣
    AIトランスフォーメーション： Gemini 2.5を始めとするマルチモーダルAIと、自律的にタスクをこなす「AIエージェント」がビジネスを革新します。
  • 2️⃣
    次世代インフラ： 推論処理に特化した新型TPU「Ironwood」が登場し、AIワークロードの効率を飛躍的に向上させます。
  • 3️⃣
    データと分析の進化： 「会話型BI」から、AIが自律的にインサイトを導き出す「エージェント型プラットフォーム」へと進化します。
  • 4️⃣
    AI時代のセキュリティ： AIを活用して防御を固める「Agentic SOC」構想により、脅威検知と対応を高度に自動化します。
  • 5️⃣
    開発者体験の革新： 開発者の90%がAIを利用。プロトタイプ自動生成など、AIが開発プロセスを根本から変えます。

4. AI時代のセキュリティ

AIが防御の中核を担う「Agentic SOC」構想

セキュリティ戦略の重点領域

Agentic SOC

AIエージェントが脅威のトリアージ、調査、対応を自動化。人間のアナリストは高度な分析に専念。

AIエージェントのセキュリティ

プロンプトインジェクションやデータ漏洩など、AI特有の脅威からエージェント自身をリアルタイムで防御。

統合セキュリティ

Security Command Centerでコンプライアンス、データセキュリティ、リスクレポートを統合管理し、全体を可視化。

IDとアクセス管理 (IAM)

Geminiが最小権限のロールを自動推奨。機密操作には再認証を要求し、不正アクセスを防止。

5. 開発者体験の革新

DORAレポートが示すAIのインパクトと開発文化の重要性

DORAレポートに見るAIのインパクト

信頼のパラドックス

多くの開発者がAIの生産性向上効果を認めつつも、その出力結果を全面的には信頼していません。AIは人間の判断を補助する強力な「副操縦士」として活用されています。

文化を映す鏡

AIは組織文化を増幅させます。結束力の高いチームでは効率をさらに高める一方、分断された組織ではその弱点を浮き彫りにします。AI導入の成功は、技術だけでなく文化に依存します。
